# Autocomplete Lab
Exploring the inner workings of autocomplete

# Installation

Clone repository to working directory
```console
$ git clone https://github.com/DPHSCodingClub/autocomplete_lab.git
cd autocomplete_lab
```
<br>

Install node requirements

```console
$ npm install
```

<br>


Run node server
```console
$ npm start
```
